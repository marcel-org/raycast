# Marcel Changelog

## [Initial Version] - {PR_MERGE_DATE}

### Added

- View and manage Marcel quests directly from Raycast
- Quest organization by status (To Do, In Progress, Completed)
- Complete quests with keyboard shortcuts (⌘ + Return)
- Change quest status via Actions menu
- Search and filter quests
- Display quest difficulty levels (easy, medium, hard, epic, legendary)
- Show XP and gold rewards for each quest
- Quest details including space, journey, date, and time
- Cached data with optimistic updates for fast UI
- Secure API token storage in Raycast preferences
- Proper error handling with user-friendly messages
- Support for custom API URL configuration
