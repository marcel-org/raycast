# Marcel for Raycast

Manage your Marcel quests from Raycast.

## Features

- View and organize quests by status
- Complete quests and earn rewards
- Change quest status
- Search functionality
- Cached data with optimistic updates

## Setup

1. Get your Marcel API token from https://marcel.up.railway.app
2. Install and configure:

```bash
npm install
npm run dev
```

3. Enter your API token in extension preferences (`⌘` + `,`)

## Usage

Open Raycast, type "View Quests", and:
- `⌘` + `Return` to complete a quest
- Use Actions menu to change status
- Search to filter quests

## Development

```bash
npm run dev          # Development mode
npm run build        # Production build
npm run lint         # Check code style
npm run publish      # Publish to Raycast Store
```
